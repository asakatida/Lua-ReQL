# Lua-ReQL

[![Build Status](https://travis-ci.org/grandquista/Lua-ReQL.svg?branch=master)](https://travis-ci.org/grandquista/Lua-ReQL)

Rethinkdb driver in Lua

## Installing
- luarocks install lua-reql

## Dev Dependencies
- Lua >= 5.1
- Luarocks
  - busted
  - dkjson [optional]
  - luajson
  - luasocket
- Python >= 3.3
- RethinkDB

## Testing
- python3 build.py

## Installing from source
- luarocks make
